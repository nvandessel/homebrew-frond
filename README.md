# homebrew-tap

Homebrew tap for packages by [@nvandessel](https://github.com/nvandessel).

## Available Casks

| Cask | Description | Repo |
|------|-------------|------|
| `floop` | Behavior learning for AI agents | [nvandessel/floop](https://github.com/nvandessel/floop) |
| `frond` | Agent-first CLI for managing stacked PRs | [nvandessel/frond](https://github.com/nvandessel/frond) |

## Usage

```bash
brew tap nvandessel/tap
brew install --cask floop
brew install --cask frond
```

## Updating

```bash
brew update
brew upgrade --cask floop frond
```
